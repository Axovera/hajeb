Thank you for using Hajeb (c) PCS from CoEIA.
	
This file will show you how to install Hajeb correctly and how to uninstall it.

System Requirments:
1- Windows 2003,XP,VISAT,2008,7.
2- Administrator Access.

-------------------------
Hajeb Installation.
-------------------------
## Login As Administrator, then:

1- Install Setup.exe
2- Run Hajeb


-------------------------
Hajeb Uninstallation.
-------------------------
## Login As Administrator, then:

In Windows Xp:
1- Run Hajeb Remover from Program start menu.
2- Remove Hajeb from Add/Remove Program.

In Windows 7:
1- Remove Hajeb from Add/Remove Program.
